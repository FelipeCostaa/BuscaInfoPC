# BuscaInfoPC

Recebe via interface grafica o Nome, Setor e Função do colaborador e em seguida faz uma verificação no sistema buscando:

 - Memória RAM total
 - Sistema Operacional
 - Modelo da CPU
 - Modelo da GPU
 - Tamanho total de armazenamento
 - Tipo de armazenamento (HDD ou SSD)

Após ser realizada a verificação ele envia para o meu e-mail todos os dados listados acima.
